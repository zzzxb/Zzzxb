一.配置Git
--------------------
*在本地创建ssh key

***$ ssh-keygen -t rsa -C "your_email@youremail.com"***

* 设置username和email

<code>$ git config --global user.name "your name"</code><br>
<code>$ git config --global user.email "your_email@youremail.com"</code>
